Documentation

We have provided a README describing the structure of this workflowr and additional README files in subfolders as needed. Briefly, this github provides all data and code needed to train DELFI-Pro models and reproduce figures. Code needed to train models is in the code folder while the code for each figure and supplementary figure is in the Analysis folder. The data folder contains data that code in Analysis relies on, and docs contains html markdown files and the produced figures.

A list of R-related dependencies for reproducing the figures in the manuscript is available in the SessionInfo document. Most of the R packages are available from Bioconductor or CRAN.

There are two packages that we wrote with useful functions - useful.stuff.aa and liver.tools. Copies of both are available in the code folder. useful.stuff.aa is originally from https://github.com/cancer-genomics/artemis2024 liver.tools is originally from https://github.com/cancer-genomics/reproduce_liver_final
